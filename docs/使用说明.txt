Windows 启动程序管理器 - 使用说明
========================================

📋 功能简介
- 查看当前开机启动项
- 添加新的启动程序
- 删除不需要的启动项
- 设置启动参数
- 打开启动文件夹

🚀 快速开始
1. 双击 StartupManager.exe 启动程序
2. 程序会自动显示当前所有启动项
3. 点击"添加程序"按钮添加新的启动项
4. 选择要删除的项目，点击"删除"按钮

💡 主要操作
【查看启动项】
- 启动程序后自动加载显示
- 显示程序名称、路径、参数、添加时间
- 点击列标题可以排序

【添加启动项】
- 点击"添加程序"按钮
- 输入程序名称
- 点击"浏览..."选择程序文件
- 可选：输入启动参数
- 点击"确定"完成添加

【删除启动项】
- 在列表中找到要删除的项目
- 点击该行的"删除"按钮
- 确认删除操作

【其他功能】
- 刷新列表：重新加载启动项
- 打开启动文件夹：直接访问系统启动文件夹

📁 启动文件夹位置
当前用户: %userprofile%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
所有用户: %ProgramData%\Microsoft\Windows\Start Menu\Programs\Startup

⚠️ 注意事项
1. 本程序只能管理用户级启动项
2. 需要普通用户权限即可运行
3. 通过快捷方式(.lnk)实现启动管理
4. 删除启动项会直接删除快捷方式文件

🔧 系统要求
- Windows 10/11
- 无需安装 .NET Runtime（独立版本）

📞 技术支持
如遇问题，请检查：
1. 程序文件是否存在
2. 是否有文件夹写入权限
3. Windows 系统版本是否支持
